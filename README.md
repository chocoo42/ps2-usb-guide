# PlayStation 2 USB Guide (FreeDVDBoot + FunTuna)

Guía completa y detallada para jugar en una PlayStation 2 utilizando un pendrive USB, mediante el exploit FreeDVDBoot y el método FunTuna.

Este proyecto documenta todo el proceso de principio a fin, incluyendo preparación en PC, instalación en la consola, configuración de Open PS2 Loader y carga de juegos desde USB.

La guía está pensada tanto para usuarios con conocimientos técnicos como para personas sin experiencia previa que simplemente quieren revivir su PS2.

---

## Contenido de la guía

- Requisitos de hardware y software
- Preparación del DVD con FreeDVDBoot
- Preparación del pendrive (FAT32)
- Instalación de FunTuna en Memory Card
- Configuración inicial de Open PS2 Loader (OPL)
- Organización correcta de juegos (estructura OPL)
- Carga de juegos desde USB
- Limitaciones del sistema FAT32 (archivos > 4 GB)
- Alternativas para juegos grandes (red / HDD)
- Errores comunes y recomendaciones prácticas
